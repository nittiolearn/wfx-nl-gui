Enter the user information and press the Create / Modify button

User type: User type determines the permission level of the user within the system.

User id: Unique id of the user within the group. Must be all lowercase without spaces or special characters. "userid.groupid" will the login id of the user.

First name: Mandatory.

Last name: Optional.

Email: Mandatory - the registration mail will be sent to this email id.

Organization unit: Mandatory. The department/user group of the user.

Secondary organization units: Optional. You could add the user to further user groups which act like distribution lists.
