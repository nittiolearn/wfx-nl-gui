Search list properties

  • name: Mandatory - provide a search list name which will help others to understand the purpose of creating the search list.
  • description:search list description - provide a short description, so help others to understand.
  • configuration:Mandatory - configuration should be in JSON format described below.

Search list configuration JSON format

Search list configuration is dictionary.

Hint: http://www.jsoneditoronline.org/ provides a good tool for editing JSON strings.

Fields "table", "filter" and "report_fields" are mandatory.

  • "table" indicates which database table to search = currently only value "lesson_approved is supported"
  • "filter" is a dictionary of table field name to filter set of values. Only records which match the filter are displayed in the search list. The criteria is AND of different filter fields and OR within the values in each field.
  • "report_fields" is list of fields within the lesson content which should be stored (and used for filtering)
  • "title" is optional title for the seach results page

{
  "title": "HBR Layout - Enghish and Science lesson plans",
  "table": "lesson_approved",
  "filter": {
    "subject": [
      "English",
      "Science"
    ],
    "center name": [
      "HBR Layout"
    ],
    "custtype": [
      "7"
    ]
  },
  "report_fields": [
    "center name",
    "year",
    "overall rating"
  ]
}
