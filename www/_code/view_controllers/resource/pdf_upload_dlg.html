Upload your content

Uploading your existing content is very easy. Save your content (documents, presentation slides, ...) as PDF file and simply upload them here.

PDF file: Select the PDF file from your device.

upload as a single page module:

Select this option to create a single page module. This will create a module with one page and all the pages of the PDF file will be displayed one below another. This is more suited for below usecases:

  • Reading/learning material
  • Flowing content (e.g. PDF file converted from a word document)
  • Large content which requires quick scrolling

Do not select this option to create a module with as many pages as the PDF file. Each page PDF file is positioned in one page of the module. This is more suited for below usecases:

  • Content where you would like to insert other pages within the module - e.g. interlacing it with some questions.
  • Slide view content (e.g. PDF file converted from a presentation slideset)

Name: This is the name of the module. Once you select the PDF file, automatically the file name is selected here. If you want to edit it, you may do so.

{{subjectLabel}}, {{gradeLabel}}: These fields help you in organizing your content.

Description, Keywords: These fields are optional - these fields helps in searching your content repository.

You could modify all the fields except the uploaded PDF file at a later point of time.
